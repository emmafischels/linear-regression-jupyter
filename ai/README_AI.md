# AI-Enhanced Linear Regression Analysis

This folder contains an AI-enhanced version of the linear regression assignment for the `linear-regression-jupyter` repository. The analysis models **Salary** from **Years of Experience** using both Python and R, with annotated plots and detailed regression diagnostics beyond the baseline manual implementation.

## Dataset

`regression_data.csv` contains two columns:

| Column | Description |
|--------|-------------|
| `YearsExperience` | Years of professional experience |
| `Salary` | Corresponding salary |

## Enhancements Over Baseline

The AI-enhanced workflow adds:

1. **Annotated regression plots** — fitted line, 95% confidence band, regression equation, R², adjusted R², and RMSE displayed on the figure.
2. **Diagnostic panels** — residuals vs fitted, normal Q-Q plot, scale-location plot, and residual histogram.
3. **Printed diagnostics** — residual summary statistics and maximum Cook's distance for influence screening.
4. **Reproducible CLI scripts** — command-line entry points mirror the notebook workflow.

## Environment Setup

From the `ai/` directory:

```bash
bash setup_env.sh
conda activate 7030_class_1
```

### Python Dependencies

- pandas, matplotlib, scikit-learn, numpy, scipy, nbconvert

### R Dependencies

- Base R, ggplot2, IRkernel (via Conda)

## Running the Notebooks

```bash
conda activate 7030_class_1
jupyter lab --no-browser --port=2000
```

Open `linear_regression_python.ipynb` (Python kernel) or `linear_regression_r.ipynb` (R kernel). Export HTML with:

```bash
jupyter nbconvert --to html linear_regression_python.ipynb
jupyter nbconvert --to html linear_regression_r.ipynb
```

## Running the Command-Line Scripts

### Python

```bash
python linear_regression_python.py regression_data.csv YearsExperience Salary
```

**Outputs:**
- `linear_regression_python_output.png` — annotated regression plot
- `regression_plot_python.png` — copy of annotated plot
- `regression_diagnostics_python.png` — four-panel diagnostic figure

### R

```bash
Rscript linear_regression_r.R regression_data.csv YearsExperience Salary
```

**Outputs:**
- `linear_regression_r_output.png` — annotated regression plot
- `regression_plot_r.png` — copy of annotated plot
- `regression_diagnostics_r.png` — four-panel diagnostic figure

## Model Results (Python)

On the provided dataset, the enhanced Python analysis reports:

| Metric | Value |
|--------|-------|
| Intercept | ~29,204 |
| Slope | ~8,285 per year of experience |
| R² | 0.7852 |
| Adjusted R² | 0.7583 |
| RMSE | ~4,186 |

These values align with the R `summary(model)` output in the companion notebook.
